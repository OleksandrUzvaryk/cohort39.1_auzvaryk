Данная папка содержит несколько различных классов, которые основаны на изображении
мобильных телефонов в теории.
Данный код пердназначен для практики наследования, где каждый новый класс
наследуется от предыдущего, дополняя и расширяя его функционал.

родительский класс -> дочерний класс

Mobile1g -> Mobile2g -> Mobile3g -> Mobile4g -> Mobile5g
